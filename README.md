# ✈️ Flight Fare Prediction using Machine Learning

## 📖 Project Description
Flight ticket prices fluctuate based on several factors such as airline, source, destination, journey date, duration, and number of stops. This project uses machine learning techniques to analyze these factors and predict flight fares accurately.

## 🎯 Objective
The objective of this project is to build a predictive model that helps estimate flight ticket prices based on historical flight data, enabling users to make informed travel decisions.

## 📂 Dataset Features
The dataset includes:
- Airline
- Date of Journey
- Source
- Destination
- Route
- Departure Time
- Arrival Time
- Duration
- Total Stops
- Additional Information
- Price (Target Variable)

## 🛠️ Technologies Used

- Scikit-learn
- XGBoost
- Jupyter Notebook

## 🔍 Project Workflow
1. Data Collection
2. Data Cleaning & Preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Flight Fare Prediction

## 🤖 Machine Learning Models
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor
- XGBoost Regressor

## 📊 Model Evaluation
The models were evaluated using:
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

The XGBoost model achieved the best prediction performance among all the models tested.
